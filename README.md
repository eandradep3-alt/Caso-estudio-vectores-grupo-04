# Caso-estudio-vectores-grupo-04
## Taller: Casos de Estudio con Vectores en JavaScript

## Descripción

Este repositorio contiene el desarrollo de dos casos de estudio realizados en JavaScript, aplicando los temas vistos en clase, como vectores (arrays), ciclos, estructuras condicionales, funciones y validaciones.

El objetivo del taller fue resolver situaciones reales mediante el desarrollo de aplicaciones web sencillas utilizando HTML y JavaScript.


## Caso 1: Simulador de Factura de Ventas

Este sistema simula la generación de una factura de ventas de una tienda de abarrotes.

### Funcionalidades

- Selección de productos.
- Ingreso de cantidades.
- Registro de productos en vectores.
- Cálculo automático del subtotal.
- Cálculo del IVA (15%).
- Aplicación del descuento del 5% cuando el subtotal supera los $20.
- Cálculo del total a pagar.
- Resumen de los productos comprados.
- Botón para limpiar todos los datos.

## Caso 2: Cajero Bancario Inteligente

Este sistema simula las operaciones básicas de un cajero bancario.

### Funcionalidades

- Registro de depósitos y retiros.
- Validación de saldo disponible.
- Almacenamiento de movimientos en vectores.
- Cálculo del total depositado.
- Cálculo del total retirado.
- Cálculo del saldo final.
- Botón para limpiar la información registrada.

## Tecnologías utilizadas

- HTML5
- JavaScript
- Visual Studio Code
- Git y GitHub
